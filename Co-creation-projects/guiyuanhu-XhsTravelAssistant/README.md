# 行程岛 · 基于小红书笔记的智能旅行助手

> 完整项目地址：https://github.com/guiyuanhu/guiyuanhu-XhsTravelAssistant
> 在线体验：https://guiyuanhu-xhs-travel-assistant.vercel.app

把你收藏的小红书旅行笔记，一键变成精美行程规划。

## 核心功能

- Bookmarklet 一键采集小红书笔记，无需复制粘贴
- 文件夹分组管理，按目的地归档
- AI 智能规划每日行程 + 种草亮点 + 避雷提醒
- 高德地图可视化，按天分色连线
- Unsplash 自动搜索景点图片
- 行程可编辑：文字修改、拖拽排序、增删景点

## 技术栈

- 前端：HTML / CSS / JavaScript（单文件）
- 后端：FastAPI + Python
- AI：DeepSeek API
- 地图：高德地图 JS API
- 图片：Unsplash API
